# 安装

*   下载下来解压

*   谷歌浏览器

    *   更多工具 -- 扩展程序（来到页面：chrome://extensions/） -- (右上角的)开发者模式打开 -- 把解压的文件直接拖到该页面

*   点开 postwoman 插件,会出现四个选项

    ```go
    接口调试
    json 工具
    官网
    系统设置（setting）
    ```

*   系统设置

    *   系统服务地址

        ```go
        默认 ： http://api.crap.cn
        改成 ：http://postwoman.cn
        
        两个服务器地址是通用的，不用进行数据迁移
        只是该成国内的速度更快
        ```

        

## 登陆

*   以 github 登陆会有问题，所以直接在web官网页面上登陆
    *   插件和官网是互通的
