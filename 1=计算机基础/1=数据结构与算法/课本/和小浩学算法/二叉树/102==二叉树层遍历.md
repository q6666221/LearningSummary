## 题目

```go
 给你一个二叉树，请你返回其按 层序遍历 得到的节点值。 （即逐层地，从左到右访问所有节点）。


// 示例：
二叉树：[3,9,20,null,null,15,7],

    3
   / \
  9  20
    /  \
   15   7

// 输出
[
  [3],
  [9,20],
  [15,7]
]
```

## 思路

```go
从输出的结果看，结果是一个二维数组，使用 res :=  [][]int{} 保存结果
其中 res[0] 是第一层 root  res[1] 是第二层 res[2]是第三层
```
