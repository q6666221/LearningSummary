## 题目

```go
给定一个字符串，找到它的第一个不重复的字符，并返回它的索引。如果不存在，则返回 -1 。

提示：你可以假定该字符串只包含小写字母

// 示例
s = "leetcode"
返回 0

s = "loveleetcode"
返回 2
```

## 分析

```go
// 方法一 
1. 由于字母共有 26 个，所以我们可以声明一个 26 个长度的数组（该种方法在本类题型很常用）

2. 因为字符串中字母可能是重复的，所以我们可以先进行第一次遍历，在数组中记录每个字母的最后一次出现的所在索引。

3. 然后再通过一次循环，比较各个字母第一次出现的索引是否为最后一次的索引。
	如果是，我们就找到了我们的目标，如果不是我们将其设为 -1（标示该元素非目标元素）
	如果第二次遍历最终没有找到目标，直接返回 -1 即可。

// 方法二
1. 创建一个map用来存放 字符和其出现的次数，找到 次数是 1 的字符
	
	但是 golang 遍历 map 的话，他是无序的，对于多个 单个字符出现的时候，我们还需要去判断哪个字符是处于最前面的
```





## golang 实现

```go
func firstUniqChar(s string) int {
    var a [26]int   // 直接使用 var a [len(s)]int
    
    // 每个字符的下标，记录下来，并及时更新
    for i , v range s {
        a[v - 'a'] = i
    }
    
    // 重新遍历 s 
    for i , v range s {
        // 看字符首次出现的小标是不是它最后的下标。是的话就是要找的哪个元素
        if i == a[v - 'a'] {
            return i
        }
    } 
}
```
