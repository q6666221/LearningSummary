

## 题目

```go
假设你正在爬楼梯。需要 n 阶你才能到达楼顶。

每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？

注意：给定 n 是一个正整数。

// 例子
输入： 3
输出： 3
解释： 有三种方法可以爬到楼顶。
1.  1 阶 + 1 阶 + 1 阶
2.  1 阶 + 2 阶
3.  2 阶 + 1 阶
```



## 分析

```go
1 个楼梯  ：1 种
2 个楼梯  ：2 种
	1阶 + 1阶
	2阶
3 个楼梯  ：3 种
4 个楼梯  ：5 种
	1 + 1 + 1 + 1
	1 + 1 + 2
	1 + 2 + 1
	2 + 1 + 1
	2 + 2

// 然后你就可以发现规律,计算当前的结果需要使用前面两步的结果
f(n) = f(n-1) +f(n-2)
```

## golang 实现

```go
// 创建一个切片，用来存储每个楼梯个数对应的结果，以后使用的时候，就可以直接调用前面的就行

func climbStair (n int) int {
    // 用来存放每个楼梯对应的方法个数
    result := make([]int,n)
    // n==1 n==2 是初始化条件，这个是需要给出的
    result[1] = 1
    result[2] = 2
    
    
    if n < 0 {
        return 0
    }
    // 只有一个楼梯
    if n == 1 {
        return result[1]
    }
    
    // 有两个楼梯
    if n == 2 {
        return result[2]
    }
	
    // 两个以上的楼梯
    for i := 3;i <= n;i++ {
        result[i] = result[i-1] + result[i-2]
    }
    
    return result[n]
}
```
