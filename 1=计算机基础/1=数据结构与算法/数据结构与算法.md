## 题目

```go
给定一个二叉树，找出其最大深度。
二叉树的深度为根节点到最远叶子节点的最长路径上的节点数。

说明: 叶子节点是指没有子节点的节点。

// 实例
给定二叉树 [3,9,20,null,null,15,7]，

    3
   / \
  9  20
    /  \
   15   7

它的最大深度 3 
```

## golang 实现

```go
/*
	type TreeNode struct {
		data int
		left *TreeNode
		right *TreeNode
	}
*/

func maxDepth(root *TreeNode) int {
    // 判断当前节点是否空，即到达了叶子结点的下面
    // 迭代终止条件
    if root == nil {
        return 0
    }
    return max(maxDepth(root.left),maxDepth(root.right)) + 1
}

func max(a,b int) int {
    if a > b {
        return a
    }
    return b
}
```
