## 超链接

```html
超链接实现页面跳转
没有独占一行 ， a 表标签中可以嵌套任何元素

<!--anchor 锚 -->
默认当前页面的打开
<a href=" www.baidu.com">  百度 </a>
<a href=" ./index.html ">  百度 </a>
<a href=" www.baidu.com" target="_blank">  百度 </a>
<a href=" # ">  回到顶部  </a>

锚点连接
回到底部就是到达最后一个标签 ，需要给最后一个标签添加标签
<a href= "#bottom" >  回到底部  </a>
<a id ="bottom"  href=" # ">  回到顶部  </a>

空连接
使用 "javascrip:;"  作为 href 的值，点击，什么也不会发生
<a href="javascrip:;"></a>



    属性
        href 指定跳转的目标路径（蓝色表示没有访问过，紫色表示访问过），
            值可以是一个外部网站地址 、
            内部页面 href = " 路径 " ，同一个目录可以直接写文件名字，
            #  返回顶部 , href="#id" 可以去任意 id 所在的任意位置
            #  在开发过程中，可以作为超链接的路径的占位符使用


        target 用来指定超链接打开的位置
            1 _self 默认值 ， 当前页面打开超链接
            2 _blank 在一个新的页面中打开超链接
        id 唯一不重复
            每一个标签都可以添加一个 id 属性，
            id 属性就是元素唯一的表示，同一个页面中不能出现重复的 id 属性


相对路径
    跳转带一个服务器内部的页面时候，我们一般使用相对路径
    ./     当前文件所在的目录 ,         当什么都不写的时候，默认 ./
    ../    当前文件所在目录的父目录 
    一个 斜杠 表示一层目录

绝对路径
```

## 图像

```html
图片标签用于向当前页面中引入一个外部图片
替换元素

<img src="./img/q.gif">
    src 指定外部图片的路径 ，分为相对路径和绝对路径，也可以直接引入网站图片网址
    alt 对图片描述 ，描述默认情况下不显示，有些浏览器会在图片加载不出来的时候显示
        搜索引擎会根据图片的 alt 中的内容来识别图片，如果不写，就不会被搜索引擎所搜索
    width 图片宽度 像素
    height 图片高度 像素
        高度或宽度如果只改变了一个。则另一个会等比例变化
        同时改变，则按照修改的参数（pc 不建议修改图片大小，需要多大切多大）


```
