## 语法规则

* 属性名称必须小写 （大小写敏感）
* 属性值必须加 引号
* 用 id 属性代替 name 属性
* DTD 定义了强制使用的 HTML 元素
* 属性不能简写

```html
<table width="100%" />
    属性名     width         小写的
    属性值     100%           加了引号
    结尾       /            前面加一个额外的空格

<img id="picture1" />
    用 id 取代 name :a applet iframe img map
```
