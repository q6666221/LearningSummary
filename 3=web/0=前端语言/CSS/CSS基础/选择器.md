## 选择器

### 简单选择器

#### 标签选择器(元素)

```css
	以html的标签名为选择器，根据标签名选中指定的元素
    语法 ： 
		标签名{}
    例子 ：
		h1{} 选中 h1 中所有的元素
```

#### 类选择器

```css
类选择器

class  （常用）
   一个标签的属性，它和 id 类型，class 可以重复使用
    class 属性来为元素分组,可以选择多个
    可以为一个元素指定多个 class 属性
语法 ：
	.属性名{}
类名字
	长命名、数组命使用中横线来命名
	不要使用纯数字、中文命名，尽量使用英文命名
例子
    <head>
 		<style>
			 .p2{
     			   color:red;
    		  }
		</style>
    </head>

<p class="red">222</p>
/*注意空格隔开
 * style 中有 .p2 .red  两个选择器样式的话，会叠加使用
* 就是设置多个 class
*/
<p class="p2 red" >222 </p>
```

#### id 选择器

```css
id 选择器
	id 只有一个
    根据元素的 id 属性值选中一个元素
    id 最好不要使用相同的，升级维护有问题
	只能调用一次，别人请勿使用

语法 ： 
	#id属性值{}

#red{
    color:red;
}


<p id="red"> 344</p>
```



#### 统配选择器

```css
通配选择器
    选中页面中的所有元素
    语法 *{}
注意
	不需要调用，自动给所有元素使用
```



### 复合选择器

```css
交集选择器
    选中同时复合多个条件的元素
    语法：选择器1 选择器2 选择器3 。。。

注意
        交集选择器中如果有元素选择器，必须使用元素选择器开头

选择 class为 red 的元素，设置字体红色

.red{
    color:red;
}

选择 class为 red 的 div 元素，设置字体大小为 30px
div.red{
    font-size:30px;
}

<div class="red" >我是 div 元素</div>
<p class="red" >  我是 p 元素</p>

多个 class 的 ： .a.b.c{}

```

### 关系选择器

```css

祖先元素	父元素		子元素		兄弟元素	相邻元素	
	跟人类的辈分一样，一级一个称谓，这是相对而言的

<p><!-- p 为li div p 的祖先元素 -->
	<ol>    父元素
		<!-- li div p 是ol 的子元素 -->
		<!-- li div p 是兄弟元素 -->
		<li></li>
		<!-- div 后相邻元素为 p， -->
		<div></div>
		<p></p>
	</ol>
</p>


包含选择器 E F  
	命中 E 的所有后代元素 F
	p li{} : 选中 p 的所有其后代元素为  li 的元素
子选择器  E>F   
	选择父E的子元素 F
	p ol{} : 选中 p 的所有儿子元素为 ol 的元素
后相邻选择器 E+F  
	选择 E 的 后相邻元素 F
兄弟选择器 E~F  
	选择 F 的兄弟(同级别)元素 F

选择器分组
	交集( , 表示 和 的意思)
		出现至少一个选择器，就会被命中
		选择器1 ， 选择器2 ， 选择器3 ， {}
	并集
		就是多个嵌套的包含选择器 ，必须都命中
```

### 属性选择器

```css
/**/
属性选择器
符号
	[]
意义
	标签[attr]  // 直接选出具有相关属性的标签
	标签[attr="value"]  /*选择属性值为 value 的标签*/
	attr ~ ="value"  
		~		有arr属性，且以 value 开头
		^		有arr属性，值为 value 
		$		有arr属性，且以 value 结尾
		*		有arr属性，值含有 value 字符串，
		|	 	拥有att属性，并且值为val，或者值是以val-开头的






比如：文章首字母下沉
伪元素
    表示页面中一些特殊的并不真实的存在的元素（特殊位置）
    伪元素使用   :: 开头
    语法
        ::first-letter 表示第一个字母
        ::first-line   表示第一行
        ::selection{}  鼠标选择的文字
    结合 content 使用
        ::before       元素的开始
        ::after          元素的结束



p::first-letter{} p 标签的第一个位置的字母
p::before{  通过before 是通过 css 添加的，无法选中
    content:abc;    在 p 标签的的最前面添加abc
}
```

### 结构伪类选择器

```css
使用
	根据文档结构来确定选择器元素，常用于父级选择器里面那选择子元素
	：表示，用于向某些选择器添加特殊的效果

父子伪类
	p:nth-child(n) // 命中孩子中 p 标签中的第 n 个
    first-child	
    last-child	
    only-child  
	/*E:nth-child(n)会选择父元素的第n个子元素E，如果第n个子元素不是E，则是无效选择符，但n会递增*/
    nth-chid(n)
	nth-last-child(n)
    -------------
    first-of-type
	last-of-type
	only-og-type
	nth-of-type(n)
	nth-last-of-type(n)

	注意
		n 可以是数字，关键字，公式
		数字 ：从 1 开始
		公式 ：第 0 个元素或者超出元素个数的会被忽略 ，从 0 开始的 ，不能是其他字母，必须是 n 
				2n 偶数  2n+1 奇数  n+5 从第5个开始 -n+5 前5个
			

连接伪类 
	:link :visited :hover :active
	a:hover{}  鼠标从 a 标签上滑过需要添加的效果
	注意
		连贯使用，需要上面四个选择器按照顺序使用 即 LVHA love hate
		给连接单独指定样式的， a 标签
	使用组合
        a {color:red;}
        a:hover {color:blue}

表单伪类
	:focus  获取焦点的表单元素 ， 焦点就是光标，一般是 input 类
	input:focus{}
	
```

### 伪元素选择器

```css
/*CSS3将伪对象选择符(Pseudo-Element Selectors)前面的单个冒号(:)修改为双冒号(::)用以区别伪类选择符(Pseudo-Classes Selectors)，但以前的写法仍然有效。*/
帮助我们利用css创建新的标签元素，而不需要HTML标签，从而简化HTML结构


伪元素
	1创建一个元素，但是属于行内元素, 新创建的这个元素在文档树中是找不到的，所以称之为伪元素
element::before{}
	在父元素内容前面创建元素
element::after
	在父元素内容后面插入元素


before after 必须有 content属性，即使空着也要写

```
