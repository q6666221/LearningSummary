## 概述

*   jQuery 是一个 JavaSript 函数库，轻量级别的写得少，做的福
*   功能
    *   HTML 元素选取、操作、事件函数
    *   HTML DOM  遍历和修改
    *   JavaSript 特效和动画
    *   AJAX
    *   Utilities
