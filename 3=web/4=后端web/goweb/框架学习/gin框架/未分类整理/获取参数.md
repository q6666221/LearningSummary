## 获取 querystring 参数

```go
querystring指的是URL中?后面携带的参数，例如：/user/search?username=小王子&address=沙河。


url中  query = 小王子 & age = 18  // 表示多个键值对，不要穿太多
1. GET 请求 URL ,后面的是 querystring 参数
	key = value 格式，多个 key-value 用 & 连接  ：ewb/query=小王子 & age=18


func main() {
	//Default返回一个默认的路由引擎
	r := gin.Default()
	r.GET("/user/search", func(c *gin.Context) {
        
        // 如果找到用你的，找不到就用指定的默认值
		username := c.DefaultQuery("username", "小王子")
 		// 获取浏览器那边发请求携带的 qurey string 参数
        //username := c.Query("username")  ，通过 query 获取请求中携带的 querystring 参数
		address := c.Query("address")
        
        // 取不到返回 false 
        message,ok  := c.GetQuery("query")
        if !ok {
        	// 取不到,使用的
            "addres" = "someaddress"
        }
        
		//输出json结果给调用方
		c.JSON(http.StatusOK, gin.H{
			"message":  "ok",
			"username": username,
			"address":  address,
		})
	})
	r.Run()
}
}
```
